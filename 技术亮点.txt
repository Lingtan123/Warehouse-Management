基于 Spring Boot + MyBatis-Plus 搭建后端服务，使用 BaseMapper、ServiceImpl、LambdaQueryWrapper 和分页插件完成用户、仓库、分类、商品及出入库记录等模块的 CRUD 与条件查询开发。

使用 MySQL 完成用户、商品、仓库、分类、记录等核心表结构设计，并结合业务需求实现基础查询与关联查询。

分页检索与多表联查封装，结合 MyBatis-Plus 分页能力与自定义 SQL 实现商品、仓库、分类、操作人等信息的多表关联查询，提升后台列表查询的可用性与开发效率。

查询功能的权限区分，普通员工仅能查询到和自己相关的记录

使用 Maven 管理项目依赖与构建流程，提升项目开发与维护的规范性。

实现基于角色的菜单控制与动态路由，登录后后端根据用户角色返回菜单数据，前端基于菜单动态挂载路由，实现不同角色访问不同功能页面。

实现库存与出入库记录联动逻辑，商品新增时同步生成记录，入库/出库时联动更新库存数量并保存操作流水，形成“商品主数据 + 业务记录”的闭环。

JWT无状态登录身份验证方案，前端用户进行访问后端操作时对其信息进行JWT加密，进行进一步身份认证和防止令牌篡改

核心功能执行的日志打印，保证出错后可以即使复现bug

加入全局异常处理器，统一接收controller调用方法可能出现的异常并统一处理

dto包封装所有请求，避免请求和实体类直接交互，降低系统的耦合度
